### Installation

1. **Navigate** to the Portfolio-Website folder on your Desktop
2. **Double-click** on `index.html` to open in your web browser, or
3. **Serve** the files using a local server:
   ```bash

   ```

4. **Visit** `http://localhost:8000` in your browser
